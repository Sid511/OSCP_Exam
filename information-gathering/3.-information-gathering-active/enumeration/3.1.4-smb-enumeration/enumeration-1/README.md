# 3.1.5.1 SMB NetBIOS Enumeration

The SMB NetBIOS34 service listens on TCP ports 139 and 445, as well as several UDP ports.

Tools such as nmap, nbtscan, etc. could be used for enumeration of NetBIOS services.
